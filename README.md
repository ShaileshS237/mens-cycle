# mens-cycle

A lightweight menstrual cycle tracker web app with:

- Cycle tracking inputs (last period start, period length, cycle length)
- Next period prediction
- Ovulation prediction (14 days before next period)
- Calendar view of period days, fertile window, and ovulation day
- Daily logs for mood, symptoms, flow, and notes
- Reminders for period and ovulation
- Email/password account-style login with encrypted local data storage

## Run

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.
